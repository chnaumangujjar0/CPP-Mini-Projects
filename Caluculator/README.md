# 🧮 Calculator (C++ OOP-Based)

A console-based **Calculator application** built using **Object-Oriented Programming (OOP)** in C++.
This project performs basic arithmetic operations with a user-friendly menu system and supports continuous calculations.

---

## 🧠 How It Works

* The user enters an initial number
* Selects an operation from the menu:

  * ➕ Addition
  * ➖ Subtraction
  * ✖️ Multiplication
  * ➗ Division
* Enters the next number
* The result is calculated and becomes the new base value
* The process continues until the user exits

---

## ✨ Features

* 🧩 OOP-based design using a `Calculator` class
* 🔁 Continuous calculations (chained operations)
* 📋 Menu-driven interface
* ⚠️ Division by zero handling
* 🔄 Option to start a new calculation
* ❌ Exit option

---

## 🛠️ Technologies Used

* C++
* Standard Library (`iostream`)

---

## ▶️ How to Run

### 🔧 Compile:

```bash
g++ main.cpp -o calculator
```

### ▶️ Run:

```bash
./calculator
```
