# 🎓 Student Management System (Console-Based in C++)

## 📌 Overview

The **Student Management System** is a console-based C++ application designed to manage student records efficiently. It allows users to perform essential operations such as adding, viewing, updating, searching, and deleting student data.

This project demonstrates practical implementation of **Object-Oriented Programming (OOP)** concepts and usage of **STL (vector)** to build a real-world system.

---

## 🚀 Features

* ➕ Add new student records
* 📋 Display all students
* 🔍 Search student by registration number
* ✏️ Update student details (Name / Marks)
* ❌ Delete student record
* 🧾 Menu-driven user interface

---

## 🧱 Project Structure

### 🔹 `Student` Class

Represents individual student data:

* Registration Number
* Name
* Marks

---

### 🔹 `StudentSystem` Class

Handles all system operations:

* Add student
* Display students
* Search student
* Update student data
* Remove student

---

## 🧠 Concepts Used

* Object-Oriented Programming (Classes & Objects)
* Constructors
* Vectors (STL)
* Loops & Conditional Statements
* Data Management
* Menu-driven program design

---

## 🔄 Program Flow

1. User selects an option from the menu
2. Performs operations like:

   * Adding a student
   * Viewing all students
   * Searching by registration number
   * Updating or deleting a record
3. Program runs continuously until user exits

---

## 💡 Example Menu

```
1. Add Student
2. Display Students
3. Search Student
4. Delete Student
5. Update Data
6. Exit
```
