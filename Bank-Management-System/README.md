# 🏦 Bank Management System (C++ Console Application)

A console-based **Bank Management System** developed in C++ using Object-Oriented Programming and STL (`vector`).
This project simulates basic banking operations such as account creation, login, deposit, withdrawal, and balance inquiry.

---

## 🧠 Project Overview

This system allows users to:

* 📝 Create a new bank account
* 🔐 Login using account number
* 💰 Deposit money
* 💸 Withdraw money
* 📊 Check account balance
* 🚪 Logout securely

The program uses a **menu-driven interface** and stores multiple accounts using a dynamic data structure.

---

## ✨ Features

* 🧩 OOP-based design using `BankAccount` class
* 📦 Multiple accounts stored using `vector`
* 🔐 Simple login authentication system
* 💰 Deposit & withdrawal operations with validation
* 📊 Real-time balance checking
* 🚫 Input validation for invalid transactions
* 🔄 Continuous system until user exits

---

## 🛠️ Technologies Used

* C++
* Standard Library:

  * `iostream` (input/output)
  * `vector` (dynamic storage)

## 🧠 How It Works

* Each user is represented as an object of `BankAccount`
* All accounts are stored in a `vector`
* The system searches accounts using account number for authentication
* Transactions update the balance dynamically
